# DevLanka Portfolio

This is a [Next.js](https://nextjs.org) project built with React and Tailwind CSS.

## Getting Started

### 1. Clone the repository
```bash
git clone <repository-url>
cd "DevLanka Portfolio"
```

### 2. Install dependencies
```bash
npm install
```

### 3. Environment Variables (If Applicable)
If local environment variables are required:
1. Create a `.env.local` file in the root directory.
2. Add your environment variables to `.env.local` (do not commit this file to Git).

### 4. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploying on Vercel

1. Push your changes to GitHub.
2. Import the repository into [Vercel](https://vercel.com).
3. If your application uses environment variables, configure them under **Project Settings > Environment Variables** in your Vercel Dashboard.
